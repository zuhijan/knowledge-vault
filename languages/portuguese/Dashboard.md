## Как добавлять новые слова
1. Добавить запись в `Words.md`
2. Если есть пример — добавить его в `Phrases.md`
3. Привязать `phrase_ids`
4. При необходимости добавить `topic_ids`

## Правила id
- для слов: `falar`, `mesa`, `comprar`
- для фраз: `falar_1`, `falar_2`, `mesa_1`
- для тем: `shopping`, `home`, `communication`
- для грамматики: `ser-vs-estar`, `presente`
